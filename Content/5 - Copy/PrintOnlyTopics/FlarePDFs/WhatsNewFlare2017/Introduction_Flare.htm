Introduction

Following are the new features introduced in this release of .

Feature

What's New?

Dynamic Preview Window Pane

  • Updates automatically when changes made in XML Editor
  • Select different file, target, or conditions
  • Lock preview in place
  • Dock window pane

Clean XHTML

  • Output format that produces content only
  • Strips out MadCap-specific tags
  • No skin, navigation, or search
  • Ideal for embedding in an application or performing post-processing tasks

Pinning Variables

  • Most recent variables shown in interface
  • Pin favorite variables to interface for easy access

Synchronizing TOC Entries with Variables

  • New variables to keep TOC entries in sync with topic title, heading, or file name
  • No need to manually change TOC label
  • Works with browse sequences too

Associating Conditions and File Tags in the Add File Dialog

  • Choose conditions and/or file tags when adding any type of file to project
  • Means you do not need to remember to apply conditions or file tags to the file later

Converting Equations to MathML from Word

  • New option when importing Word documents
  • Lets you convert equations to MathML format (the web standard)
