Adding a Master Page to a Project

how to add a master page to a project

    • (Recommended) Right-Click In the Content Explorer, right-click a folder and from the context menu select New>Master Page.

    • Ribbon In the Content section select New>Add Master Page.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Master Page is selected.

  5. In the File name field, type a new name for the master page.
  6. (Optional)   See Applying Conditions to Content.
  7. (Optional)   See Associating Tags with Files.
  8. Click Add. The master page file is added. The XML Editor opens to the right, with the new master page shown.
